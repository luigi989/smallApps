## :microscope: Apps

### :seedling: Install and launch
1. Execute the following command in the folder apps/:
```
  npm start
```
2. The website will be shown in your standard web browser.

### :ballot_box_with_check: Prerequisites
- Node 16.13.0 or later
- Npm 8.1.0 or later
